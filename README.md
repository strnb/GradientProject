# GradientProject: Клиент-серверное приложение (Вариант №2)

Данный проект представляет собой многопользовательскую систему для выполнения защищенных математических расчетов. Проект выполнен в рамках учебного курса «Технологии программирования».

---

##  Основной функционал

Проект разделен на три ключевых блока:

### 1. Криптография и безопасность
* **Шифр Виженера:** Реализован алгоритм симметричного шифрования для защиты передаваемых данных.
* **SHA-256:** Используется для безопасного хранения паролей в базе данных и проверки целостности пакетов.
* **Авторизация:** Система ролей (Админ / Клиент) с проверкой прав доступа.

### 2. Математические вычисления
* **Интерполяция сплайнами:** Построение гладких кривых по набору входных точек.
* **Градиентный спуск:** Оптимизация и поиск минимума целевой функции.

### 3. Архитектура
* **Client-Server:** Поддержка нескольких одновременных подключений.
* **Singleton Pattern:** Реализовано одиночное подключение к БД и управление клиентом.
* **GUI:** Оконный интерфейс для визуализации графиков и управления данными.

##  Структура проекта и Git

Соблюдается стандартная структура веток:
* `main` — стабильная версия.
* `develop` — основная ветка разработки.
* `feature/crypto` — разработка алгоритмов шифрования.
* `feature/math` — реализация математических методов.
